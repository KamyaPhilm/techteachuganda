Python Programming Language

Python is a high-level programming language, with applications in numerous areas, including web programming, scripting, scientific computing, and artificial intelligence. Many companies need developers so there are many job opportunities when you learn programming.

Python communicates with computers. You specify a series of instructions that will control the behavior of the computer. A program is made by joining those instructions.

Enter the wonderful world of programming

Introduction

In this program you learn the basics of programming in Python. Python is an accessible programming language with many different applications. For example, you can use Python to build a website such as a webshop. But Python is also very suitable for programming Internet of Things (IOT) devices or controlling hardware.

In this program you mainly learn the basics of Python, but also a good basis for learning a different programming language because the concepts that are discussed can be used in many other programming languages.

Python is used for:
  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.
Why Python?
  • Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • Python has a simple syntax similar to the English language.
  • Python has syntax that allows developers to write programs with fewer lines.
  • Python runs on an interpreter system, meaning that code can be executed as soon as it is written. This means that prototyping can be very quick.
  • Python can be treated in a procedural way, an object-orientated way or a functional way.
Good to know
  • The most recent major version of Python is Python 3, though we shall be using Python 2 in the tutorial. However, Python 2, although not being updated with anything other than security updates, is still quite popular.
  • In the tutorial Python will be written in online editor. It is possible to write Python in an Integrated Development Environment, such as Thonny, Pycharm, Netbeans or Eclipse which are particularly useful when managing larger collections of Python files.
